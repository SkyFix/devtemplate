#Gulp шаблон для разработки front-end
````
Get started ?
$ npm i - установит все необходимые модули и зависимости для проекта
$ npm run start - создает и инициализирует сетку из миксинов, копирует файлы(библиотеки)
...........................и запускает проект в режиме разработки
$ npm run favicon создает фавикон для сайта или веб приложения
$ npm run build запускает проект в режиме production(минимизирует и оптимизирует проект)
$ npm run stylelint запускает линтер для файлов scss и фиксит некоторые синаксические ошибки
$ npm run eslint запускает линтер для файлов js и фиксит некоторые синаксические ошибки
